# nma_changing_things

[Reading material for project](https://github.com/pravsels/nma_changing_things/reading.md)

## Setup

Build docker container:
```
chmod u+x *.sh

./docker_build.sh
```

Run the container:
```
./run_docker_container.sh
```

## Run

Run sample script: 
```
python random_actions.py 
```
